Para tener una mejor ortografía y redacción sigue estos consejos:

// No escribas como hablas.
// No le temas al punto seguido. En internet es mejor usar los párrafos cortos y claros en vez de párrafos muy largos y repletos de comas.
// Lee en voz alta lo que escribiste.
// Dale estructura a tu texto. Las negritas te pueden ayudar a resaltar contenido importante que podría perderse entre todos los párrafos.
// Consigue un amigo editor.
// Lee mucho.

Complementación

Usa siempre una herramienta de corrección de texto (Google docs o Microsoft Word te sirven)
además, te dejo estos tips que yo tengo en un blod de notas y siempre que tengo una duda lo consulto:

Halla: Verbo Encontrar -> halla la solución

Haya: Verbo haber -> Lo que haya hecho

Alla: Lugar -> Alla en rusia

Hay: Verbo Haber -> No hay nada acá

Alli: Lugar -> Alli en la cocina

Ay: Dolor -> Ay! me duele

A ver: Verbo Vista -> Vamos a ver eso

Haber: Tener -> Parece haber alguien

ves: Verbo Ver -> vamos a ver eso

vez: veces -> una vez vi eso

Ha: Verbo haber -> Ha encontrado algo

A: Indica lugar/tiempo -> A esa persona

Ah: Expresar pena -> Ah que pena con ustedes
